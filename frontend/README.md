# Insurance Frontend

## Setup
```bash
cd frontend
npm install
cp .env.example .env # fill in addresses
npm run dev
```

Env:
- `VITE_POLICY_ADDRESS`, `VITE_TREASURY_ADDRESS`, `VITE_MOCK_ORACLE_ADDRESS` (optional locally)
- `VITE_EXPLORER_BASE` like `https://sepolia.etherscan.io`

Go to `/` to open the Policies page.
